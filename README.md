# exponential-idle-theory-automation-QoL
Combined QOL and TA script for exponential idle.

- will auto theories assumed to use theorysim algorithms for optimal publication multipliers
- has QOL enhancements to auto allocate starts + students
- can automatically turn off R9 and reallocate students into other variables based on two conditions:
  - either right before prestige condition, or
  - when DB is increasing less than 1e/10 seconds.

  This allows for R9 swapping if db increase rate is too low to reach last prestige condition.
  AutoR9 might not occour if DB increase is too rapid due to supremecy, which is mostly inconsequential.
